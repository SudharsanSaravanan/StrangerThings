# Stranger Things 

This is a web application created for Stranger Things die-hard fans.
The website allows users to watch all seasons and episodes of Stranger Things in one place.

The project uses the **VidKing API** to stream episodes and provides a simple and clean interface for browsing and watching content.


## About the Project

* All seasons and episodes available
* Easy navigation between seasons and episodes


## Disclaimer

This project is created for educational and fan purposes only.
It is not affiliated with Netflix or the official Stranger Things franchise.
All content rights belong to their respective owners.

## Only for Laptop/Desktop Viewers

This website supports movie and episode viewing only on laptops and desktops.
Mobile viewing is not supported. Please use a laptop to watch online.
